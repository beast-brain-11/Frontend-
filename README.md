## Prerequisites

- Node.js (v16 or higher)
- npm or yarn
- Modern web browser

## Getting Started

1. Clone the repository:
```bash
git clone [repository-url]
cd 100x
```

2. Install dependencies:
```bash
cd frontend/project
npm install
# or
yarn install
```

3. Start the development server:
```bash
npm run dev
# or
yarn dev
```

4. Open your browser and navigate to `http://localhost:5173`
